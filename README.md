## Prerequisites & Setup

### Requirements
- [Docker](https://docs.docker.com/get-docker/) >= 29.3.0
- [Python](https://www.python.org/downloads/) >= 3.12.1
- [Git](https://git-scm.com/)
- A [Google Cloud Platform](https://cloud.google.com/) account (free tier is sufficient)

### Clone the repository
```bash
git clone https://github.com/robert-sanz03/data-engineering-zoomcamp-2026.git
cd data-engineering-zoomcamp-2026
```
